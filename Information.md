¡Hola! Bienvenido al creador de Servidores para OptiCraft!

Como usarlo:
- Inicia el "Start-Server.bat"
- Espera a que se termine de generar el mundo. Puedes darte cuenta cuando las letras en el CMD no se muevan
- Abre OptiCraft en la version 1.7 - 1.9 - 1.12.
- Ve al apartado de "Jugar"
- Ve al apartado de "Servidores"
- Añade un nuevo servidor con los siguientes datos:
NOMBRE: (El que tu quieras)
IP: localhost
PUERTO: 19132
- Unete al servidor y listo! ya puedes jugar!

Añadir plugins:
- Ve a la carpeta de "plugins"
- Descomprime el archivo de "plugins.7z"
- Inicia el servidor y veras la recopilacion de Plugins funcionando!

Descargar e importar plugins custom:
- En tu navegador, pon esta url: https://cloudburstmc.org/resources/categories/nukkit-plugins.1/
- Busca el plugin que tu quieras
- Toca el boton de "Download"
- Cuando se te descargue el archivo ".jar" simplemente copialo a la carpeta "Plugins"
- Inicia el servidor y veras el plugin probablemente funcionando!

Hostearlo a multiples personas utilizando Playit:
- Descarga playit: https://playit.gg/download
- Selecciona el ejecutable para tu sistema operativo, en mi caso hare el ejemplo con Windows.
- Toca el boton de "Download"
- Ahora, ejecuta el programa e instalalo
- Una vez hecho esto, inicia el programa
- Cuando lo hagas, seguramente te salga una URL en un CMD, simplemente copiala seleccionandola y tocando Ctrl + C
- Pega la URL en el navegador
- Create una cuenta y logeate
- Ahora espera a que el programa de Playit se comunique con el sitio web
- Si te sale un cartel de "Add new agent" toca continue.
- Toca "Add Agent"
- Espera nuevamente a que se comunique
- Una vez comunicado, toca la opcion de "Create Tunnel"
- Aqui, selecciona la region mas cercana a ti y elige la opcion de "Minecraft Bedrock (game)"
- Toca "Add tunnel"
- Una vez hecho eso abre un CMD
- Escribe "ipconfig" y toca "Enter"
- Busca donde dice "IPv4 Address"
- Copia la IP utilizando Ctrl + C 
- Regresa al navegador y a la pagina de Playit
- Cambia la IP de 127.0.0.1 a la tuya, en mi caso es 192.168.100.254
- Toca "Update"
- Copia la IP que se encuentra en la parte de arriba en la pagina
- Repite los pasos de "Como usarlo" Remplazando la IP y el puerto por el de Playit, puedes enviarle la IP a multiples usuarios.
- Listo! Si tienes fallos recuerda desactivar tu FireWall!
